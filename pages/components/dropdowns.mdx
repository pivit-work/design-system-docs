# Dropdowns

드롭다운은 서브뷰에서 액션을 그룹화할 때 사용합니다.
메뉴를 숨기거나 여러 액션을 하나로 묶어야 할 때 유용합니다.

Figma 컴포넌트: `Base components/Dropdowns`

---

## Dropdown Trigger

드롭다운을 여는 트리거 버튼입니다. `Button` / `Icon` / `Avatar` 세 가지 타입을 지원합니다.

### Default

<ComponentPreview code={`<Dropdown trigger="button" label="Account" />
<Dropdown trigger="icon" />
<Dropdown trigger="avatar" avatarText="JL" />`}>
  <Dropdown trigger="button" label="Account" />
  <Dropdown trigger="icon" />
  <Dropdown trigger="avatar" avatarText="JL" />
</ComponentPreview>

### Open (focus ring)

`open` 상태에서는 브랜드 컬러의 포커스 링이 나타납니다.

<ComponentPreview code={`<Dropdown trigger="button" label="Account" open />
<Dropdown trigger="icon" open />
<Dropdown trigger="avatar" avatarText="JL" open />`}>
  <Dropdown trigger="button" label="Account" open />
  <Dropdown trigger="icon" open />
  <Dropdown trigger="avatar" avatarText="JL" open />
</ComponentPreview>

## Select

폼에서 사용하는 선택 입력 컴포넌트입니다. 라벨에 옵션 `*` 표시(필수)와 헬프 아이콘을 포함할 수 있고, 값 옆에 보조 텍스트(`@핸들`)가 인라인으로 표시됩니다.

### 기본 Select

<ComponentPreview direction="column" code={`<Select label="Team member" required value="John Lee" supportingText="@john" hint="This is a hint text to help user." />
<Select label="Team member" placeholder="Select team member" />`}>
  <Select label="Team member" required value="John Lee" supportingText="@john" hint="This is a hint text to help user." />
  <Select label="Team member" placeholder="Select team member" />
</ComponentPreview>

### 사이즈

<ComponentPreview direction="column" code={`<Select label="Size sm" value="John Lee" supportingText="@john" size="sm" />
<Select label="Size md" value="John Lee" supportingText="@john" size="md" />`}>
  <Select label="Size sm" value="John Lee" supportingText="@john" size="sm" />
  <Select label="Size md" value="John Lee" supportingText="@john" size="md" />
</ComponentPreview>

### Leading 변형

<ComponentPreview direction="column" code={`<Select label="Icon" value="John Lee" supportingText="@john" leadingIcon={<MailIcon />} />

  <Select label="Icon" value="John Lee" supportingText="@john" leadingIcon={<MailIcon />} />

</ComponentPreview>

## Figma Variants

### Dropdown

| Property | Values |
| --- | --- |
| **Type** | `Button` `Icon` `Avatar` |
| **Open** | `True` `False` |

### Dropdown List Item

| Property | Values |
| --- | --- |
| **Icon** | `True` `False` |
| **Checkbox** | `True` `False` |
| **State** | `Default` `Hover/active` `Disabled` |
| **Divider** | `True` `False` |

### Select

| Property | Values |
| --- | --- |
| **Size** | `sm` `md` |
| **Type** | `Default` `Icon leading` `Avatar leading` `Dot leading` `Search` `Tags` |
| **State** | `Default` `Placeholder` `Focused` `Disabled` `Open` |

| Property | Values |
